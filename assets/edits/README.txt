Shop + product carousel images (from ~/Downloads/edits)
=======================================================
Only images from this folder are used for the shop top carousel.
Copy your images from:
  ~/Downloads/edits
into this folder and name them:
  edit1.png, edit2.png  (and edit3.png, etc. if you have more)

Use .jpg instead of .png if needed; the page will try .jpg if .png fails.

- Shop carousel (top of shop): uses edit1, edit2 in pages/shop.html.
- Product page carousel: uses edit1, edit2 in pages/product-magazine.html.

To add more slides: add more <div class="shop-carousel-slide"> in shop.html
and/or more <div class="carousel-slide"> in product-magazine.html.
